**Screens**
- **Onboarding / Login:** Email/password, social login, forgot password.
- **Dashboard:** Overview with projects, active tasks, analytics widgets.
- **Project View:** Project details, task lists (Backlog, In Progress, Done), filters, and sorting.
- **Task Detail:** Title, description, comments, attachments, subtasks, assignees, due date, status, history.
- **Create/Edit Task:** Form with fields for title, description, labels, priority, due date, assignee, attachments.
- **Boards / Kanban:** Drag-and-drop columns, quick actions on cards.
- **Search & Global Filter:** Search results, saved filters, sort options.
- **Notifications / Inbox:** Activity feed and read/unread state.
- **Profile / Settings:** User preferences, account settings, integrations.

**User Flows (top-level)**
- **Create task flow:** Dashboard → +Create → Fill form → Save → Notification + Project list update.
- **Assign & notify flow:** Task assign → Notify assignee → Assignee sees in Inbox → Accept/act.
- **Move to done flow:** Drag card to Done → Update task status and completed timestamp → Trigger webhook (optional).

**Design Tokens (core)**
- **Colors:** Primary, Secondary, Surface, Background, Success, Warning, Danger, Text Primary, Text Secondary.
- **Spacing:** base 8px scale (4,8,16,24,32,40,48).
- **Typography:** H1 (48/56), H2 (32/40), H3 (24/32), Body (16/24), Small (12/16). Use system stack with fallback.
- **Radii:** small 4px, medium 8px, large 12px.
- **Shadows:** elevation-1, elevation-2 (subtle usage for cards and modals).

**Reusable Components**
- **Button:** Primary, Secondary, Tertiary (states: default, hover, pressed, disabled).
- **Input:** Text, textarea, select, datepicker, file upload, with validation states.
- **Card / Task Card:** Title, meta row (assignee, labels, due date), actions menu, drag handle.
- **Modal:** Centered, dismissible, with optional confirmation actions.
- **Badge / Label:** Color-coded, supports icons and counts.
- **Avatar:** Circular, 24/32/40px sizes.
- **Toolbar / Topbar:** App nav, search, global actions, user menu.
- **Toast / Notification:** Auto-dismiss and persistent forms for important actions.

**Accessibility Notes**
- Use semantic HTML roles for dialogs, lists, forms.
- Ensure 4.5:1 contrast for body text; 3:1 for large text.
- Keyboard support for Kanban drag/drop (use move-to shortcuts) and form submission.
- Provide ARIA labels for interactive elements and descriptive alt text for images.

**Assets & Exports**
- Icon set: export as SVG (optimized), 24px and 16px variants.
- Illustrations: PNG / WebP for production; SVG for scalable use when possible.
- Logos: SVG and PNG (dark/light variants).
- Spacing & layout examples: include CSS snippets for grid and container widths.
